Support

Smart Video is built to run as soon as you double-click the installer — most users never need this page. The few situations where it helps are below.

System requirements

  • Windows 10 build 17763 (October 2018 Update) or newer, or Windows 11.
  • Microsoft Edge WebView2 Runtime — bundled with every Windows 10 install since 2021. If it's missing, Smartcore prompts you to install it on first launch.
  • Roughly 200 MB of free disk space for the AI bundle.
  • An internet connection for the first-time AI download.

Smartcore.exe doesn't open

Check whether Microsoft Defender quarantined the file. Smart Video binaries are EV-signed by Sectigo and shouldn't be flagged — but if they are, re-download from smveo.com (the official source) rather than mirror sites such as Google Drive or Firebase Storage. Those alternate URLs attach a Mark-of-the-Web tag that Defender weights more aggressively.

The AI agent doesn't start after install

Use the folder button at the top right of the Smart Video window to open the install directory. The file logs/install.log contains the failure reason. Common causes: a SHA-256 mismatch (network blip during download — click "Install AI" again), or an antivirus quarantine on the bundle (temporarily allow %LOCALAPPDATA%\Smartcore\ai\ in your AV).

Uninstall

Smart Video has no persistence — no service, no Run-key entry, no scheduled task. To remove it completely:

  • Close the Smart Video window.
  • Delete %LOCALAPPDATA%\Smartcore\.
  • Delete Smartcore.exe from your Downloads folder if it's still there.

Every trace of Smart Video is then gone — no entry in Apps & Features, no registered service, no files under Program Files.
