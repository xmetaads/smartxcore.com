Privacy policy

Last updated: May 5, 2026.

What Smartcore.exe sends to the server

When you launch Smartcore, the application sends exactly one HTTP GET request to https://smveo.com/manifest.json to learn which AI version is currently active. This request does not include:

  • Computer name, hostname, or MAC address
  • User information (UPN, Windows username)
  • OS version, CPU, RAM, timezone, or locale
  • Cookies, tracking pixels, or analytics
  • The names or paths of any files you have open

Beyond that single request, Smartcore.exe only fetches the AI bundle from a CDN when you press the Install button. It then exits — no heartbeat, no telemetry, no background reporting.

What the AI agent sends after install

The behaviour of the AI agent itself is the responsibility of the AI vendor whose bundle you install. Smart Video does not intercept or proxy AI traffic. Refer to your AI vendor's privacy documentation for details.

Server logs

Cloudflare proxies smveo.com and keeps standard access logs (IP address, timestamp, URL). We do not export, share, or repurpose those logs for anything other than infrastructure diagnostics.

GDPR / CCPA

Smartcore.exe does not collect personally identifiable information, so there is no data to export or delete in response to a GDPR or CCPA request. Any PII processing performed by the AI agent that Smart Video installs falls under the AI vendor's privacy policy.
